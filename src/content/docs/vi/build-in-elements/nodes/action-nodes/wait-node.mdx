Action node này tạm dừng thực thi của behavior tree tại vị trí của nó trong một khoảng thời gian cố định.

**Đường dẫn trong Editor:** `Action > Wait`

### Thuộc tính

| Thuộc tính | Type  | Mô tả                                                |
| ---------- | ----- | ---------------------------------------------------- |
| `duration` | float | Thời gian tính bằng giây để chờ trước khi trả về `Success`. |

### Hành vi

-   **On Enter:** Node ghi lại thời gian hiện tại.
-   **On Process:**
    -   **Trả về `Running`** nếu thời gian đã trôi qua ít hơn `duration`.
    -   **Trả về `Success`** một khi thời gian đã trôi qua vượt quá `duration`.

### Ví dụ sử dụng
Sử dụng node này trong một `Sequence` để làm AI tạm dừng sau khi đến patrol point trước khi di chuyển đến điểm tiếp theo. (`Move To Patrol Point` -> `Wait` -> `Next Patrol Point`)

<TabItem label="C# Code">
```csharp
using UnityEngine;

namespace ND_BehaviorTree
{
    [NodeInfo("Wait", "Action/Wait", true, false,iconPath:"Assets/ND_BehaviorTree/NDBT/Icons/Wait.png")]
    public class WaitNode : ActionNode
    {   
        public float duration = 1.0f;
        [HideInInspector]
        protected float _startTime;

        protected override void OnEnter()
        {
            _startTime = Time.time;
        }

        protected override Status OnProcess()
        {
            if (Time.time - _startTime > duration)
            {   
                return Status.Success;
            }
            return Status.Running;
        }
    }
}
```
