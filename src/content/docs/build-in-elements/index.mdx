This section details all the ready-to-use nodes that come with the ND_BehaviorTree framework. These nodes are the fundamental building blocks for creating complex AI behaviors.

They are categorized by their function within the tree.

	<Card title="Action Nodes" icon="rocket">

        [See all Action Nodes](./nodes/action-nodes/)
	</Card>
	<Card title="Composite Nodes" icon="puzzle">
		The branches of the tree that control the flow of execution among their children.
        [See all Composite Nodes](./nodes/composite-nodes/)
	</Card>
	<Card title="Auxiliary Nodes" icon="substack">
		Wrapper nodes (Decorators) and background tasks (Services) that modify or support other nodes.

	</Card>
</CardGrid>
