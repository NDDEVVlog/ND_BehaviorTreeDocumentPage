Action node này ra lệnh cho `NavMeshAgent` di chuyển đến destination `Vector3` tĩnh. Điều này lý tưởng cho các tuyến tuần tra nơi các điểm không di chuyển.

**Đường dẫn trong Editor:** `Action > Movement > MoveToPatrolPoint`

### Thuộc tính

| Thuộc tính       | Type | Mô tả                                                          |
| ---------------- | ---- | --------------------------------------------------------------- |
| `agentKey`       | Key  | Blackboard key cho `NavMeshAgent` component trên AI.            |
| `destinationKey` | Key  | Blackboard key (của type `Vector3`) cho điểm destination.      |

### Hành vi

-   **On Enter:** Lấy `NavMeshAgent` và `Vector3` destination, sau đó đặt destination của agent.
-   **On Process:**
    -   **Trả về `Failure`** nếu agent bị thiếu từ Blackboard.
    -   **Trả về `Success`** khi agent đến destination.
    -   **Trả về `Running`** trong khi agent đang tính toán path hoặc di chuyển.
-   **On Exit:** Gọi `agent.ResetPath()` để dừng di chuyển của agent.

<Aside type="caution">
Node này yêu cầu `NavMeshAgent` component trên AI của bạn và NavMesh đã được bake trong scene để hoạt động chính xác.
</Aside>

```csharp
// --- START OF FILE MoveToPatrolPointNode.cs ---
using UnityEngine;
using UnityEngine.AI;

namespace ND_BehaviorTree
{
    [NodeInfo("Move To Patrol Point", "Action/Movement/MoveToPatrolPoint", true, false)]
    public class MoveToPatrolPointNode : ActionNode
    {
        [Tooltip("The Blackboard key for the NavMeshAgent component on the AI.")]
        public Key agentKey;
        [Tooltip("The Blackboard key (of type Vector3) for the destination point.")]
        public Key destinationKey;
        private NavMeshAgent agent;

        protected override void OnEnter()
        {
            if (agentKey != null && blackboard != null) agent = blackboard.GetValue<NavMeshAgent>(agentKey.name);
            if (agent == null) return;
            
            if (destinationKey != null && blackboard != null)
            {
                Vector3 destination = blackboard.GetValue<Vector3>(destinationKey.name);
                agent.SetDestination(destination);
            }
        }

        protected override Status OnProcess()
        {
            if (agent == null) return Status.Failure;
            if (agent.pathPending) return Status.Running;
            if (agent.remainingDistance <= agent.stoppingDistance) return Status.Success;
            return Status.Running;
        }

        protected override void OnExit()
        {
            if (agent != null && agent.hasPath) agent.ResetPath();
        }
    }
}
```
