Decorator này đảo ngược kết quả của child của nó. `Success` trở thành `Failure`, và `Failure` trở thành `Success`.

**Đường dẫn trong Editor:** `Decorator > Inverter`

### Hành vi

-   Nếu child của nó trả về `Success`, Inverter trả về **`Failure`**.
-   Nếu child của nó trả về `Failure`, Inverter trả về **`Success`**.
-   Nếu child của nó trả về `Running`, Inverter trả về **`Running`**.

### Ví dụ sử dụng
Hữu ích trong một `Sequence` khi bạn muốn tiếp tục chỉ khi một điều kiện nhất định *không* được đáp ứng. Ví dụ: `Inverter (Has Target) -> Wander`. Nhánh này sẽ chỉ chạy nếu AI *không* có target.

<TabItem label="C# Code">
```csharp
using UnityEngine;

namespace ND_BehaviorTree
{   
    [NodeInfo("Inverter", "Decorator/Inverter", true, true,iconPath:"Assets/ND_BehaviorTree/NDBT/Icons/Inverter.png")]
    public class InverterNode : DecoratorNode
    {
        protected override Status OnProcess()
        {
            if (child == null) return Status.Success;
            var status = child.Process();
            switch (status)
            {
                case Status.Success:
                    return Status.Failure;
                case Status.Failure:
                    return Status.Success;
                case Status.Running:
                    return Status.Running;
            }
            return Status.Failure;
        }
    }
}
```
