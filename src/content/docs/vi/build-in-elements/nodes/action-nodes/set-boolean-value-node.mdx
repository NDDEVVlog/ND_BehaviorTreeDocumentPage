Đây là một Action node đơn giản nhưng mạnh mẽ đặt giá trị boolean cho một key được chỉ định trên Blackboard. Nó rất cần thiết để quản lý trạng thái AI.

**Đường dẫn trong Editor:** `Action > SetBooleanValueNode`

### Thuộc tính

| Thuộc tính       | Type    | Mô tả                                                     |
| ---------------- | ------- | ----------------------------------------------------------- |
| `conditionKey`   | Key     | Blackboard key (của type `bool`) để đặt.                  |
| `ValueToSet`     | bool    | Giá trị boolean (`true` hoặc `false`) để gán cho key.     |

### Hành vi

-   **Trả về `Success`** sau khi đặt giá trị trên Blackboard.
-   **Trả về `Failure`** nếu `conditionKey` không được gán.

### Ví dụ sử dụng
Sử dụng để quản lý các trạng thái như `isAlerted`. Khi AI tìm thấy player, một `Sequence` có thể chạy `FindTarget` rồi `SetBooleanValue` (`isAlerted` = `true`). Sau đó, một `CheckConditionVariable` decorator có thể kiểm tra flag này.

<TabItem label="C# Code">
```csharp
// --- START OF FILE SetBooleanValueNode.cs ---
using UnityEngine;
using ND_BehaviorTree;
using Key = ND_BehaviorTree.Key;

[NodeInfo("SetBooleanValueNode", "Action/SetBooleanValueNode", true, false,iconPath:"Assets/ND_BehaviorTree/NDBT/Icons/boolean.png")]
public class SetBooleanValueNode : ActionNode
{
    [BlackboardKeyType(typeof(bool))]
    [Tooltip("The Blackboard key (of type bool) to set value.")]
    public Key conditionKey;
    public bool ValueToSet;
   
    protected override Status OnProcess()
    {   
         if (conditionKey == null) return Status.Failure;
        blackboard.SetValue<bool>(conditionKey.keyName,ValueToSet);
        return Status.Success;
    }
}
```
